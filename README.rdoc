== Vanderbilt University Residential Education

This is the independent study of Zach Roth for CS 240 during the Spring 2013
semester at Vanderbilt University. The purpose of this study is to build and
deploy a web app for ResEd to serve as a accesible portal for students to 
learn about things to do in Nashville.
